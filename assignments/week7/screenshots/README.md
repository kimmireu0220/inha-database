## 테스트 시나리오

### 테스트 1: 로그인 성공
1. 로그인 페이지 접속 (`localhost:3000/`)
2. 학번: `1`, 비밀번호: `010-0000-0001` 입력
3. 로그인 성공 → 학생 정보 조회 페이지로 이동

### 테스트 2: 학생 정보 조회
- 로그인한 학생의 정보만 표시됨
- 전체 테이블 정보도 함께 표시 (Building, Department, Room, Class)

### 테스트 3: 수강취소 기능
1. 수강취소 페이지 접속 (`localhost:3000/delete/class`)
2. 수강 중인 수업 목록 확인 (Database, Data Structures)
3. Database 수업의 "수강취소" 버튼 클릭
4. 수강취소 완료 → Data Structures 수업만 남음

### 테스트 4: 로그인 실패
1. 로그인 페이지에서 잘못된 정보 입력 (학번: `999`, 비밀번호: `wrong-password`)
2. 로그인 실패 알림 창 표시
3. 알림 창 닫기 → 로그인 페이지로 돌아옴
